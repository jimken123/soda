# LoadFileType
The type of load file to create when running an export.


|Name          |Summary                 |
|:-------------|:-----------------------|
|None          |No load file is created.|
|Discover      |Discover                |
|Ringtail      |Ringtail (MDB)          |
|Concordance   |Concordance             |
|Summation     |Summation               |
|DiscoveryRadar|Discovery Radar         |
|DocuMatrix    |DocuMatrix              |
|EDRMXML       |EDRM XML                |
|EDRMXMLZIP    |EDRM v1.2 XML ZIP       |
|IPRO          |IPRO                    |
|XHTMLSummary  |XHTML summary report    |
|CSVSummary    |CSV summary report      |
|MD5Digest     |MD5 digest report       |
