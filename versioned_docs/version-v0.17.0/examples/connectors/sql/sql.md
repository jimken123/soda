# SQL Connector Examples

## Setup

1. Download and set up Sequence. See the [quick start](../../../quick-start.md) for more info.
2. Add the SQL connector. See [connectors](../../../quick-start.md#connectors)
   for more info on managing connectors.

```powershell
./sequence.exe connector add Sequence.Connectors.Sql
```
