# Microsoft365 Steps

| Step                                                                 | Summary                                                                                 |
| :------------------------------------------------------------------- | :-------------------------------------------------------------------------------------- |
| [M365ChannelMessagesRead](./Microsoft365/M365ChannelMessagesRead.md) | Reads M365 Channels. Uses `/teams/{teamId}/channels/{channelId}/messages` endpoint.     |
| [M365ChannelsRead](./Microsoft365/M365ChannelsRead.md)               | Reads M365 Channels. Uses `/teams/{teamId}/allChannels` endpoint.                       |
| [M365ChatsRead](./Microsoft365/M365ChatsRead.md)                     | Read User mail from Microsoft 365. Uses `/Chats` endpoint.                              |
| [M365GetToken](./Microsoft365/M365GetToken.md)                       | Gets the M365 Token that was used to login.                                             |
| [M365Login](./Microsoft365/M365Login.md)                             | Login to M365. You do not need to do this.                                              |
| [M365MailRead](./Microsoft365/M365MailRead.md)                       | Read User mail from Microsoft 365. Uses `/me/mailFolders('{Folder}')/messages` endpoint |
| [M365TeamsRead](./Microsoft365/M365TeamsRead.md)                     | Read a list of users from Microsoft 365. Uses `/me/joinedTeams` endpoint                |
| [M365UsersRead](./Microsoft365/M365UsersRead.md)                     | Read a list of users from Microsoft 365. Uses `/users` endpoint                         |
