# StructuredData Steps

|Step                                                 |Summary                                                                                               |
|:----------------------------------------------------|:-----------------------------------------------------------------------------------------------------|
|[FromConcordance](./StructuredData/FromConcordance.md)|Extracts entities from a Concordance stream. The same as FromCSV but with different default values.   |
|[FromCSV](./StructuredData/FromCSV.md)                |Extracts entities from a CSV stream. The same as FromConcordance but with different default values.   |
|[FromIDX](./StructuredData/FromIDX.md)                |Create an entity from an IDX Stream                                                                   |
|[FromJson](./StructuredData/FromJson.md)              |Extracts the entity from a Json stream containing a single entity.                                    |
|[FromJsonArray](./StructuredData/FromJsonArray.md)    |Extracts entities from a Json stream containing an array of entities.                                 |
|[FromXml](./StructuredData/FromXml.md)                |Extracts the entity from a Xml stream containing a single entity.                                     |
|[FromXmlArray](./StructuredData/FromXmlArray.md)      |Extracts entities from a Xml stream containing an array of entities.                                  |
|[FromYaml](./StructuredData/FromYaml.md)              |Extracts the entity from a Yaml stream containing a single entity.                                    |
|[ToConcordance](./StructuredData/ToConcordance.md)    |Write entities to a stream in concordance format. The same as ToCSV but with different default values.|
|[ToCSV](./StructuredData/ToCSV.md)                    |Write entities to a stream in CSV format. The same as ToConcordance but with different default values.|
|[ToIDX](./StructuredData/ToIDX.md)                    |Write an entity to a stream in IDX format.                                                            |
|[ToJson](./StructuredData/ToJson.md)                  |Writes an entity to a stream in JSON format                                                           |
|[ToJsonArray](./StructuredData/ToJsonArray.md)        |Write entities to a stream in Json format.                                                            |
|[ToXml](./StructuredData/ToXml.md)                    |Writes an entity to a stream in XML format                                                            |
|[ToXmlArray](./StructuredData/ToXmlArray.md)          |Write entities to a stream in Xml format.                                                             |
|[ToYaml](./StructuredData/ToYaml.md)                  |Writes an entity to a stream in YAML format                                                           |
