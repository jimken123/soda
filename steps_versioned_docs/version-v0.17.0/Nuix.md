# Nuix Steps

|Step                                                                     |Summary                                                                                                                                                                                                                                                                                                                                                                                      |
|:------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|[NuixAddConcordance](./Nuix/NuixAddConcordance.md)                        |Adds data from a Concordance file to a NUIX case.                                                                                                                                                                                                                                                                                                                                            |
|[NuixAddItem](./Nuix/NuixAddItem.md)                                      |Adds a file or directory to a Nuix Case.                                                                                                                                                                                                                                                                                                                                                     |
|[NuixAddToItemSet](./Nuix/NuixAddToItemSet.md)                            |Run a search query in Nuix and add all items found to an item set. Will create a new item set if one doesn't already exist.                                                                                                                                                                                                                                                                  |
|[NuixAddToProductionSet](./Nuix/NuixAddToProductionSet.md)                |Run a search query in Nuix and add all items found to a production set. Will create a new production set if one with the given name does not already exist.                                                                                                                                                                                                                                  |
|[NuixAnnotateDocumentIdList](./Nuix/NuixAnnotateDocumentIdList.md)        |Annotates a document ID list to add production set names to it.                                                                                                                                                                                                                                                                                                                              |
|[NuixAssertPrintPreviewState](./Nuix/NuixAssertPrintPreviewState.md)      |Checks the print preview state of the production set.                                                                                                                                                                                                                                                                                                                                        |
|[NuixAssignCustodian](./Nuix/NuixAssignCustodian.md)                      |Run a search query on a nuix case and assign found items to a custodian.                                                                                                                                                                                                                                                                                                                     |
|[NuixCloseConnection](./Nuix/NuixCloseConnection.md)                      |Close the connection to nuix. Also closes all cases                                                                                                                                                                                                                                                                                                                                          |
|[NuixCountItems](./Nuix/NuixCountItems.md)                                |Returns the number of items matching a particular search term                                                                                                                                                                                                                                                                                                                                |
|[NuixCreateCase](./Nuix/NuixCreateCase.md)                                |Creates a new case and opens it.                                                                                                                                                                                                                                                                                                                                                             |
|[NuixCreateIrregularItemsReport](./Nuix/NuixCreateIrregularItemsReport.md)|Creates a list of all irregular items in a case. The report is in CSV format. The headers are 'Reason', 'Path' and 'Guid' Reasons include 'NonSearchablePDF','BadExtension','Unrecognised','Unsupported','TextNotIndexed','ImagesNotProcessed','Poisoned','Record','UnrecognisedDeleted','NeedManualExamination', and 'CodeTextFiles' Use this inside a WriteFile step to write it to a file.|
|[NuixCreateNRTReport](./Nuix/NuixCreateNRTReport.md)                      |Create a case report using an NRT file.                                                                                                                                                                                                                                                                                                                                                      |
|[NuixCreateReport](./Nuix/NuixCreateReport.md)                            |Creates a report for a Nuix case. The report is in csv format. The headers are 'Custodian', 'Type', 'Value', and 'Count'. The different types are: 'Kind', 'Type', 'Tag', and 'Address'. Use this inside a WriteFile step to write it to a file.                                                                                                                                             |
|[NuixCreateTermList](./Nuix/NuixCreateTermList.md)                        |Creates a list of all terms appearing in the case and their frequencies. The report is in CSV format. The headers are 'Term' and 'Count' Use this inside a WriteFile step to write it to a file.                                                                                                                                                                                             |
|[NuixDoesCaseExist](./Nuix/NuixDoesCaseExist.md)                          |Returns whether or not a case exists.                                                                                                                                                                                                                                                                                                                                                        |
|[NuixExcludeDecryptedItems](./Nuix/NuixExcludeDecryptedItems.md)          |Excludes password protected items once they have been decrypted. This only excludes encrypted original items that have a decrypted child with the same name.                                                                                                                                                                                                                                 |
|[NuixExportConcordance](./Nuix/NuixExportConcordance.md)                  |Exports a production set in Concorfance format                                                                                                                                                                                                                                                                                                                                               |
|[NuixExtractEntities](./Nuix/NuixExtractEntities.md)                      |Extract Entities from a Nuix Case.                                                                                                                                                                                                                                                                                                                                                           |
|[NuixGeneratePrintPreviews](./Nuix/NuixGeneratePrintPreviews.md)          |Generates print previews for items in a production set.                                                                                                                                                                                                                                                                                                                                      |
|[NuixGetAuditedSize](./Nuix/NuixGetAuditedSize.md)                        |Calculate the total audited size for a case. For this to work, the case material needs to have been ingested with the calculateAuditedSize=true processing setting. Nuix uses 1000 base for kb/mb/gb, not 1024.                                                                                                                                                                              |
|[NuixGetCaseDetails](./Nuix/NuixGetCaseDetails.md)                        |Returns an Entity with information about the current case. Includes Name, Description, GUID, Investigator, InvestigationTimeZone, Users, Custodians, Tags, Languages, Location (path), and if available the earliest and latest dates.                                                                                                                                                       |
|[NuixGetItemProperties](./Nuix/NuixGetItemProperties.md)                  |A step that the searches a case for items and outputs the values of item properties. The report is in CSV format. The headers are 'Key', 'Value', 'Path' and 'Guid' Use this inside a WriteFile step to write it to a file.                                                                                                                                                                  |
|[NuixGetLicenseDetails](./Nuix/NuixGetLicenseDetails.md)                  |Returns an Entity with information about the Nuix license currently in use. Includes license type, description, exprity, number of available workers, and the available features. The features list can be used when configuring the Nuix Connector.                                                                                                                                         |
|[NuixGetVersion](./Nuix/NuixGetVersion.md)                                |Returns the Nuix version                                                                                                                                                                                                                                                                                                                                                                     |
|[NuixImportDocumentIds](./Nuix/NuixImportDocumentIds.md)                  |Imports the given document IDs into this production set. Only works if this production set has imported numbering.                                                                                                                                                                                                                                                                           |
|[NuixMigrateCase](./Nuix/NuixMigrateCase.md)                              |Migrates a case to the latest version if necessary.                                                                                                                                                                                                                                                                                                                                          |
|[NuixOpenCase](./Nuix/NuixOpenCase.md)                                    |Migrates a case to the latest version if necessary.                                                                                                                                                                                                                                                                                                                                          |
|[NuixOpenConnection](./Nuix/NuixOpenConnection.md)                        |Open the connection to nuix. Does not open the case.                                                                                                                                                                                                                                                                                                                                         |
|[NuixPerformOCR](./Nuix/NuixPerformOCR.md)                                |Performs optical character recognition on items in a NUIX case.                                                                                                                                                                                                                                                                                                                              |
|[NuixRemoveFromItemSet](./Nuix/NuixRemoveFromItemSet.md)                  |Removes particular items from a Nuix item set.                                                                                                                                                                                                                                                                                                                                               |
|[NuixRemoveFromProductionSet](./Nuix/NuixRemoveFromProductionSet.md)      |Removes particular items from a Nuix production set.                                                                                                                                                                                                                                                                                                                                         |
|[NuixReorderProductionSet](./Nuix/NuixReorderProductionSet.md)            |Reorders and renumbers the items in a production set.                                                                                                                                                                                                                                                                                                                                        |
|[NuixSearchAndExclude](./Nuix/NuixSearchAndExclude.md)                    |Run a search query in Nuix and exclude all items found.                                                                                                                                                                                                                                                                                                                                      |
|[NuixSearchAndTag](./Nuix/NuixSearchAndTag.md)                            |Searches a NUIX case with a particular search string and tags all files it finds.                                                                                                                                                                                                                                                                                                            |
|[NuixRunScript](./Nuix/NuixRunScript.md)                                  |Run an arbitrary ruby script in nuix. It should return a string.                                                                                                                                                                                                                                                                                                                             |
