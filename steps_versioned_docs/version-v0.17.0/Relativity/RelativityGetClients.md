# RelativityGetClients
_Alias_:`RelativityGetClients`

_Output_:`Array<T>`

Retrieve a list of the available clients that you can associate with a matter.

## Examples
#### SCL
```scl
RelativityGetClients
```
#### Expected Output
```scl
[(Name: "Client 1" ArtifactID: 1), (Name: "Client 2" ArtifactID: 2)]
```
