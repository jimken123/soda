# NuixAddItem
_Alias_:`NuixAddItem`, `NuixAdd`, `NuixImportItem`

_Output_:`Unit`

*Requires Nuix.Version 5.0*

*Requires Nuix.Features: CASE_CREATION*

Adds a file or directory to a Nuix Case.
