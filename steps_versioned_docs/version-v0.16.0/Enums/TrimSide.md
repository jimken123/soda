# TrimSide
The side of the string to trim.


|Name |Summary                                          |
|:----|:------------------------------------------------|
|Start|Removes whitespace from the start of the string. |
|End  |Removes whitespace from the end of the string.   |
|Both |Removes whitespace from both sides of the string.|
