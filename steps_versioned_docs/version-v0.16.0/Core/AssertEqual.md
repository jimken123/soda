# AssertEqual
_Alias_:`AssertEqual`

_Output_:`Unit`

Asserts that two objects are equal. Both objects must have the same type.


|Parameter|Type|Required|Position|Summary                     |
|:--------|:--:|:------:|:------:|:---------------------------|
|Left     |`T` |✔       |1       |The first object to compare |
|Right    |`T` |✔       |2       |The second object to compare|
