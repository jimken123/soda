# SetVariable
_Alias_:`SetVariable`

_Output_:`Unit`

Sets the value of a named variable.


|Parameter|Type          |Required|Position|Summary                          |
|:--------|:------------:|:------:|:------:|:--------------------------------|
|Variable |`VariableName`|✔       |1       |The name of the variable to set. |
|Value    |`T`           |✔       |2       |The value to set the variable to.|
