# Pwsh Steps

|Step                                             |Summary                                                                                                                                                                                                                           |
|:------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|[PwshRunScript](./Pwsh/PwshRunScript.md)          |Executes a powershell script.                                                                                                                                                                                                     |
|[PwshRunScriptAsync](./Pwsh/PwshRunScriptAsync.md)|Executes a powershell script and returns any results written to the pipeline as an entity stream. Running a script asynchronously allows for input streaming. Note that the script only runs when the output of this Step is read.|
