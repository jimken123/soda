# Severity
Log severity


|Name       |Summary                                                      |
|:----------|:------------------------------------------------------------|
|Debug      |Verbose log messages that can be used for debugging SCL.     |
|Information|Log messages that track the general flow of a sequence.      |
|Warning    |Log messages for non-terminating errors or unexpected events.|
|Error      |Log messages that indicate sequence failure.                 |
