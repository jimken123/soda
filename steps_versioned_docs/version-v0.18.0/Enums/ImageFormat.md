# ImageFormat
The format of the image to OCR.


|Name   |Summary                                                 |
|:------|:-------------------------------------------------------|
|Default|Image format not specified - this may result in failure.|
|Png    |Portable Network Graphics                               |
|Bmp    |Bitmap image file                                       |
|Tif    |Tag Image File Format - TIFF or TIF                     |
|JPG    |Joint Photographic Experts Group - JPG or JPEG          |
|GIF    |Graphics Interchange Format                             |
